Installation de Oracle virtual box

L'installation de Oracle qui est un logiciel de virtualisation permet par la suite de configurer notre machine virtuelle en choisisant l'OS de notre machine pous notre cas c'est debian puis aussi de choisir les carastéristiques de la machine (vue dans la partie introduction).

Ajout des users et du groupe

Pour faire cela il faut se connecter avec l’utilisateur administrateur (« root ») ou un utilisateur ayant des privilèges sudo avec le “su -” . La commande useradd permet de crée un utilisateur, la commande groupadd permet de cree un groupe et pour finir la commande usermod –g permet d’ajouter des utilisateurs a un groupe.

Verification

La photo qui suit nous permet de vérifier que tous les users sont crées.

La commande id suivie d'un nom d'user permet de voir ses informations on peut ici l'utiliser pour vérifier si les utilisateurs crée sont bien dans le groupe sae-systèmes.
