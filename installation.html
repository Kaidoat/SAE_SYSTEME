Installation de Oracle virtual box

L'installation de OracleVirtual Box, qui est un logiciel de virtualisation nous permet par la suite de configurer notre machine virtuelle en choisissant le système d'exploitation de notre machine, pour notre cas le choix s'est porté sur Debian. La suite est de choisir les caractéristiques de la machine (vue dans la partie introduction).

Ajout des users et du groupe

Pour faire cela, il faut se connecter avec l’utilisateur administrateur (« root ») ou un utilisateur ayant des privilèges sudo avec le "su -". La commande "useradd" permet de créer un utilisateur, la commande "groupadd" permet de créer un groupe et pour finir la commande "usermod –g" permet d’ajouter des utilisateurs à un groupe.

Verification

La photo qui suit nous permet de vérifier que tous les users soient crées.

La commande "id" suivie d'un nom d'utilisateur permet de voir ses informations, nous pouvons ici l'utiliser pour vérifier si les utilisateurs sont bien crées et ajoutés dans le groupe SAE-Systèmes.

Ajout de SSH

Nous avons de plus ajouter à notre Machine Virtuelle SSH, ou Secure Shell, qui est un protocole réseau utilisé pour établir une connexion sécurisée entre deux systèmes, généralement un client et un serveur. Cela nous a permis de pourvoir faire des copiés coller sur notre machine virtuelle car nous entrons les commandes sur le CMD de notre machine physique.
