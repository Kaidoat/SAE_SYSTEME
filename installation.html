Installation de Oracle VirtualBox

L'installation de Oracle VirtualBox, qui est un logiciel de virtualisation, nous permet par la suite de configurer notre machine virtuelle en choisissant le système d'exploitation de notre machine. Pour notre cas, le choix s'est porté sur Debian. La suite est de choisir les caractéristiques de la machine (vue dans la partie introduction).

Ajout des utilisateurs et des groupes

Pour faire cela, il faut se connecter avec l’utilisateur administrateur (« root ») ou un utilisateur ayant des privilèges sudo avec la commande su -. La commande useradd permet de créer un utilisateur, la commande groupadd permet de créer un groupe, et pour finir, la commande usermod –g permet d’ajouter des utilisateurs à un groupe.

Vérification

La photo qui suit nous permet de vérifier que tous les utilisateurs soient créés.

La commande id suivie d'un nom d'utilisateur permet de voir ses informations. Nous pouvons ici l'utiliser pour vérifier si les utilisateurs sont bien créés et ajoutés dans le groupe SAE-Systèmes.

Ajout de SSH

Nous avons également ajouté à notre machine virtuelle SSH, ou Secure Shell, qui est un protocole réseau utilisé pour établir une connexion sécurisée entre deux systèmes, généralement un client et un serveur. Cela nous a permis de pouvoir copier et coller sur notre machine virtuelle en saisissant les commandes sur le terminal de notre machine physique.
